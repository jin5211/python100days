"""
ファイル操作
テキストファイルが対象(.txt)

ファイルオブジェクト
f = open(ファイルパス、モード)
モード
'r' = 読み込み(デフォルト)
'w' = 書き込み
'a' = 追記
'r+' = 読み込み + 書き込み

ファイルをopenする
with open(ファイルパス、モード) as 変数名:
  # ファイル操作
変数名のところにファイルオブジェクトが代入される(f がよく使われる)
with block を抜けるとファイルが close される。

ファルの読み込み
with open(ファイルパス) as f:
  s = f.read()

ファイルを1行ずつ読み込み
with open(ファイルパス) as f:
  for _ in range(3):
    s = f.readline()

読み込んでリストで取得
with open(ファイルパス) as f:
  for _ in range(3):
    s = f.readlines()

データ書き込み
with open(ファイルパス, 'w') as f:
  f.write(書き込みたい文字列) -> 改行は'\n'で表す

データを追記で書き込む
with open(ファイルパス, 'a') as f:
  f.write(書き込みたい文字列) -> 改行は'\n'で表す

リストを書き込む(文字列として結合され、1行でまとめて書き込んでくれる)
x = ['a', 'b', 'c']
with open(ファイルパス, 'w') as f:
  f.writelines(x)

その他 -> csv, json, xml, excel(これらは専用モジュールを使った方が良い)別の動画
"""
